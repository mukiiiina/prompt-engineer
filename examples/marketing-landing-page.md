- **Primary CTA color**: #6366F1 (indigo) — used sparingly, only for main actions
- **Secondary CTA**: Transparent bg, 1px border #6366F1, text #6366F1
- **Form fields** (if any): Email input for newsletter, minimal 1-field form

## 4. Animations

- **Hero entrance sequence**:
  - 0ms: Eyebrow fades in (opacity 0→1, 400ms)
  - 200ms: Headline fades in + translateY(20px→0), 600ms, easeOutExpo
  - 400ms: Subheadline fades in, 600ms
  - 600ms: CTA buttons fade in, 400ms
  - 800ms: Social proof fades in, 400ms
  - 1000ms: Hero image scales in (0.95→1) + fades in, 800ms
- **Scroll reveals**: Each section fades in + translateY(30px→0), 700ms, cubic-bezier(0.16, 1, 0.3, 1), triggered at 15% viewport intersection
- **Number counters**: Stats animate counting up when scrolled into view (duration 2s, easeOut)
- **Hover micro-interactions**:
  - Buttons: scale(1.03), brightness(1.1), transition 200ms
  - Cards: translateY(-4px), shadow increase, transition 250ms
  - Links: color shift + underline reveal from left, 200ms

## 5. SEO & Performance

- **Meta title**: "CodeGuard AI — Automated Code Review for Engineering Teams"
- **Meta description**: "Catch bugs, security issues, and style violations before they reach production. AI-powered code review that works in 30 seconds."
- **Open Graph image**: 1200×630, product screenshot + headline
- **Twitter card**: Summary large image
- **Schema markup**: SoftwareApplication JSON-LD
- **Core Web Vitals targets**:
  - LCP: <2.0s
  - FID: <50ms
  - CLS: <0.1
- **Performance optimizations**: Next.js Image component, lazy load below-fold sections, font-display: swap, preload critical fonts

## 6. Tech Stack

- **Framework**: Next.js 14 (App Router) + React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion (entrance sequences, scroll reveals)
- **Icons**: Lucide React
- **Fonts**: Inter (Google Fonts, weights 400/500/600/700)
- **Deployment**: Vercel
- **Analytics**: Vercel Analytics + Plausible (privacy-friendly)
- **A/B Testing**: Vercel Flags (test headline variations, CTA copy)

## 7. Responsive Behavior

- **Desktop (>1280px)**: Full layout, 3-column grids, side-by-side hero, all animations
- **Laptop (1024-1280px)**: Slightly reduced padding, 2-column feature grid
- **Tablet (768-1024px)**: Stacked layouts, 2-column pricing, hamburger nav
- **Mobile (<768px)**: Single column, reduced hero padding (100px top), simplified animations, sticky CTA bar at bottom

## 8. A/B Testing Hooks

Design with these testable elements:
- **Headline variants**: "Ship code faster" vs "Catch bugs before production" vs "AI code review in 30 seconds"
- **CTA copy**: "Start free trial" vs "Try for free" vs "Get started — 14 days free"
- **Social proof placement**: Above fold vs below hero image
- **Pricing highlight**: Monthly vs Annual (save 20%) toggle
- **Testimonial format**: Quote cards vs video testimonials
